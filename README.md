# Ear Audio Teleprompter

## Project Description
The Ear Audio Teleprompter is an innovative application designed to assist speakers by displaying text audibly. This application allows users to prepare and deliver speeches with confidence, ensuring smooth communication and clarity of thought.

## Features
- **Audio Display**: Converts text to audio so speakers can follow along seamlessly.
- **User-Friendly Interface**: Easy to navigate, making it accessible for all users.
- **Customizable Settings**: Users can adjust speed, pitch, and volume of the audio for a personalized experience.
- **Text Management**: Simple options to load, save, and edit scripts.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/GenZredeemer/Ear_Audio_Teleprompter.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Ear_Audio_Teleprompter
   ```
3. Install the required dependencies:
   ```sh
   npm install
   ```

## Usage
- To start the application, run:
  ```sh
  npm start
  ```
- Load your script, adjust settings, and press play to start the audio display.
